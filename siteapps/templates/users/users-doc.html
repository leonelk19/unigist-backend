User Endpoints

The https:/api/v1/ endpoint provides the following functionality related to user accounts:

Register a User

  • POST /api/v1/auth/register - Register a new user
    • Request Body
      • username (string, required) - The desired username for the new account
      • email (string, required) - The email address for the new account
      • password (string, required) - The desired password for the new account

Login a User

  • POST api/v1/auth/login/ / - Log in an existing user
    • Request Body
      • username (string, required) - The email address of the user account to log in
      • password (string, required) - The password of the user account to log in
    • Response
      • token (string) - Simple authentication token to use for future requests

Logout a User

  • POST /api/v1/auth/logout/ - Log out the current user
    • Request Headers
      • Authorization (string, required) - The authentication token obtained from logging in
    • Response
      • message (string) - A message indicating that the user has been logged out

Retrieve User Account Information

  • GET /api/v1/user/account - Retrieve the current user's account information
    • Request Headers
      • Authorization (string, required) - The authentication token obtained from logging in
    • Response

Update User Account Information

  • PUT /api/v1/user/account - Update the current user's account information
    • Request Headers
      • Authorization (string, required) - The authentication token obtained from logging in
    • Request Body
      • username (string, optional) - The new username for the user account
      • email (string, optional) - The new email address for the user account
      • password (string, optional) - The new password for the user account
    • Response

Delete User Account

  • DELETE /api/v1/user/account - Delete the current user's account
    • Request Headers
      • Authorization (string, required) - The authentication token obtained from logging in
    • Response
      • message (string) - A message indicating that the user account has been deleted
